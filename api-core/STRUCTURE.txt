API Core — SOILSUL (negócio)
============================

Next.js API routes: app/api/*
Prisma, auth JWT, obras, sondagem, relatórios.

Docker: api-core/Dockerfile

Comunicação:
  Browser → gateway:8080/api/* → api-core:3001
  app-web (dev) reescreve /api/* → API_CORE_URL (default http://localhost:3001)
